## Print Settings

### Rods
- **Orientation**: Vertical (standing up)
- **Brim**: 5mm (small contact area)
- **Layer**: 0.2mm
- **Infill**: Any (solid anyway)
- **Walls**: 3

### Connectors
- **Orientation**: Flat on bed
- **Layer**: 0.2mm
- **Infill**: 40-50%
- **Walls**: 3-4
- **Supports**: None needed

### Mounts
- **Orientation**: Flat on bed
- **Layer**: 0.2mm
- **Infill**: 30-40%
- **Walls**: 3
- **Supports**: Some may need

## Material
- **Recommended**: PETG-CF (carbon fiber)
- **Alternative**: PETG, ABS
- **Nozzle**: Hardened steel 0.4mm (for CF)
- **Temp**: 240-250°C
- **Bed**: 70-80°C

## Assembly
1. Print all parts (or selected plates)
2. Insert rods into connector sockets (friction fit)
3. If too loose, add drop of CA glue
4. If too tight, sand rod end slightly

## Tolerance Adjustment
If fit is too tight/loose for your printer:
- Too tight: Scale rod diameter to 4.9mm
- Too loose: Scale socket holes to 5.1mm
